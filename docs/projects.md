# プロジェクト一覧

M5Stack CoreMP135を活用した様々なIoTシステム開発プロジェクトです。

## 🚀 現在進行中のプロジェクト

### MP135 Gateway (C++)

C++による高性能IoTゲートウェイ実装プロジェクトです。

- **言語**: C++
- **特徴**: 低レベル最適化による高速処理、リアルタイムデータ処理
- **ユースケース**: 高速センサーデータ収集、産業用制御システム

[📂 GitHubリポジトリを見る](https://github.com/tinayla696/mp135_gateway_cpp){ .md-button .md-button--primary target="_blank" }

---

### MP135 Gateway (Go)

Goによるスケーラブルなゲートウェイ実装プロジェクトです。

- **言語**: Go
- **特徴**: 並行処理による高スループット、シンプルな保守性
- **ユースケース**: クラウド連携、大規模データ収集

[📂 GitHubリポジトリを見る](https://github.com/tinayla696/mp135_gateway_go){ .md-button .md-button--primary target="_blank" }

---
